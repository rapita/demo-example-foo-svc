# Demo Example Foo service

Simple GRPC + Gateway service generated by buf.

Requirements:
* [podman](https://podman.io/) (for generate code by buf)
* [golang](https://go.dev/) ^1.7
* [make](https://www.gnu.org/software/make/)

Default addresses:
* :6565 - GRPC
* :8080 - Gateway

Run:
```shell
go run cmd/main.go
```

Generate code:
```shell
make generate
```
